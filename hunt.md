# 墙后追杀 · 像素对决

> 单文件 HTML5 像素动作游戏,关羽骑战马追杀墙后乞丐,桌面/手机/微信通用。

## 🎮 文件清单

```
~/Desktop/追杀游戏/
├── index.html   ← 整个游戏(68KB,自包含 HTML+CSS+JS)
└── README.md    ← 本说明
```

**零依赖**:不需要 Python、不需要 Node、不需要数据库、不需要服务器。**一个 HTML 文件,扔到任何静态托管平台即可发布**。

---

## 🚀 发布(最简方法)

### 桌面端
直接把 `index.html` 拷到任何位置(桌面/U盘/网盘),**双击** → 默认浏览器(Chrome/Safari)打开就能玩。

### 云平台发布(推荐)
把 `index.html` 拖到以下任一平台,**1 分钟上线**:

| 平台 | 操作 |
|---|---|
| **Cloudflare Pages** | 登录 → Create → Direct Upload → 拖入 `index.html` → 完成,得到 `xxx.pages.dev` 链接 |
| **GitHub Pages** | 新建仓库 → 提交 `index.html` → Settings → Pages → 选 main 分支 → 完成 |
| **Netlify Drop** | 打开 [app.netlify.com/drop](https://app.netlify.com/drop) → 拖入 `index.html` → 完成 |
| **Vercel** | `npx vercel` → 完成 |
| **腾讯云/阿里云 OSS** | 上传到静态网站托管 → 绑定域名 |

发布后任何人用浏览器(包括微信)打开链接都能玩,**完全不需要 WiFi 局域网**。

### 微信里直接玩
发布到云平台拿到链接(比如 `https://xxx.pages.dev/`),微信里点开就能玩。

⚠️ 微信扫码本地 IP(`http://192.168.x.x`)已经不需要了——直接打开云平台链接即可。

## 📖 关于"用户登录"

**这是单机游戏,没有账号系统、没有云存档**——不需要登录。

游戏进度只存在浏览器内存里,关掉标签页就没了。

**多用户方式**:同一台电脑开多个标签页就是独立游戏;不同设备打开同一个云链接各自玩(没有联机)。

## ⚔ 游戏内容

### 两关卡模式

| 关卡 | 玩家 HP | 敌人 HP | 狗伤害 | 名称 |
|---|---|---|---|---|
| **第 1 关** | 500 | 500 | 50 | 关羽初阵 |
| **第 2 关** | 1000 | 1 亿 | 100 | 千里走单骑 |

- 第一关速战速决,打死 500 血乞丐**直接跳第二关**(无任何过渡画面)
- 第二关挑战极限:1 亿血,需要靠 buff + 长矛 + 刀盾狗持续磨
- 中途任何关卡死亡 → 按 R 回到第 1 关

### 7 个技能(5 秒冷却)

| 键 | 名称 | 玩家伤害 | 范围 |
|---|---|---|---|
| 1 | 飞镖 | 10 | 直线,30% 寒冰减速 |
| 2 | 震地 | 14 | 周围 90px |
| 3 | 如来神掌 | 35 | 落敌人位置,无距离限制 |
| 4 | 火火箭 | 16 | 追踪 5 秒,自动避墙 |
| 5 | 分身 | 5-10/只 ×5-10 | 漂移近战 10 秒 |
| 6 | 回血 | +20 HP | - |
| 7 | 瞬移 | 200px 跳跃 | - |

### 自动系统

| 系统 | 规则 |
|---|---|
| **长矛**(关羽专属) | 每 5 秒自动发 1 发。打中敌人=10 伤;打中自己分身=生成 5 个新分身 |
| **空投** | 每 20 秒,落在 1 秒前目标位置(±5px),拾取 +15 HP |
| **千里马** | 每 5 分钟闪现 1 秒(纯视觉特效) |
| **刀盾狗**(只打敌人) | 100 HP,撞人 50 伤/次。击杀 → +15 HP + 5-15 攻击 buff(3 秒) |
| **刀刃狗** | 刀盾狗技能变出,5-10 只,撞人 50/100 伤(按关卡) |

## 📱 移动端 + 微信适配

- **iOS Safari / Android Chrome**:dpad + 7 技能按钮自动显示,完美触控
- **微信内置浏览器**:X5 内核,首次触摸会激活音频(已测试兼容)
- **屏幕缩放**:`viewport` 锁定,禁止双击放大
- **iPhone 刘海屏**:`env(safe-area-inset-*)` 兼容小白条
- **小屏手机**(<420px):技能按钮字号自动缩小,防溢出
- **横竖屏切换**:自动适应,横屏布局最佳

### 触屏特有的反馈
- 按钮按下时背景变亮(`.pressed` 类)
- 方向键按住持续触发移动,松开立即停止
- 释放兜底:iOS 有时 touchend 不触发,已加全局 touchcancel/blur 兜底
- 阻止长按弹出菜单、双击放大、滑动穿透

## 🔧 技术细节

- **纯 Canvas 2D** + ES6,所有角色/敌人/狗用代码绘制(零图片资源)
- **Web Audio API** 合成音效(被敌人分身撞到 → 短促蜂鸣)
- **静音开关**:触屏右下角 🔊音 按钮可一键静音
- **多分辨率适配**:dpr 缩放、aspect-ratio 保持比例、小屏字号自适应
- **iOS 安全区适配**:`env(safe-area-inset-*)`

## 📝 维护说明

如果要改数值,所有关卡参数都在 `index.html` 第 ~110 行 `LEVELS` 对象:

```js
const LEVELS = {
  1: { playerHp: 500,    enemyHp: 500,       dogDmg: 50,  label: '关羽初阵' },
  2: { playerHp: 1000,   enemyHp: 100000000, dogDmg: 100, label: '千里走单骑' },
};
```

直接改这个对象里的数字,Chrome 刷新即可。
