## 一句话总结

**"工具没有行业限制，普通人都能用 AI 解决自己工作里每天重复出现的问题"**——日本北海道农民 Hiroki Tomiyasu 用 Codex 把 1500 亩农场的"病虫害识别 / 卫星遥感监测 / 温室 IoT 控制 / 客服机器人"四件事从手工/外包流程改造成了"ChatGPT 学习 + Codex 落地"的个人 AI 工作台；这是 OpenAI 官方案例对"非技术行业 AI 落地"的最强背书。

## 关键观点

1. **农业 AI 落地全景案例（Hiroki Tomiyasu，OpenAI 官方案例）**
   - 100 公顷 / 1500 亩 / 洋葱+小麦+甜菜
   - 没有软件公司背景 / 不懂 AI 原理 / 没有技术团队
   - 关键不是他有多厉害，而是**他想到了，他做了**

2. **病虫害多模态识别**
   - 拍照片上传 → AI 自动分析叶片状态 + 病虫害特征
   - 替代"自己判断原因 / 请专家过来看"
   - 节省"面积上百公顷农场主"的大量时间和精力

3. **卫星遥感 + AI 主动监测**
   - 把卫星遥感数据接进系统
   - 农场每块地的**长势 / 颜色变化 / 水分**自动监测
   - 某个区域异常 → 系统**主动提醒**（"巡田"被 AI 接管）

4. **IoT 温室远程控制**
   - Codex 帮自己开发了温室控制系统
   - 手机上看**温度 / 湿度 / 设备状态**
   - 远程**调整参数 / 控制大棚卷帘升降**

5. **农业知识库客服机器人**
   - 农场内部的聊天机器人
   - 回答**设备操作 / 流程执行 / 作物注意事项**
   - 农民遇到问题不用翻手册 / 不用到处问人

6. **ChatGPT vs Codex 的"学习/干活"二分**
   - ChatGPT = 思考、做问答、查资料、学习新知识
   - Codex = 具体干活（按你说的把东西做出来）
   - **从 ChatGPT 到 Codex 是从"问"到"做"的升级**

7. **"先 ChatGPT 学习，再 Codex 落地"模式（普通人 AI 使用路径）**
   - 步骤 1：通过 ChatGPT 学习新知识、查资料、解决问题
   - 步骤 2：通过 Codex 把想法真正做出来
   - 这是一条"非程序员也能用"的 AI 工作流

8. **工具低估 / 行业 AI 落地反思**
   - 研发思维 vs 普通用户思维的鸿沟
   - "工具出来之后，它并没有局限什么行业、什么人可以用。它更像一种新的生产力"
   - **真正重要的问题不是"AI 会什么"，而是"我工作里哪些事可以交给它"**
   - 反问：**"你的工作里，有没有一个每天重复出现、却一直没人解决的问题？"**
   - "下一个被 Codex 改造的行业，不是什么高科技行业。而是你所在的那个行业。"

## 4 个具体 AI 应用场景

| 场景 | 技术栈 | 替代了什么 |
|---|---|---|
| 病虫害识别 | 多模态视觉 AI | 专家判断 |
| 卫星遥感监测 | 卫星数据 + 异常检测 | 人工巡田 |
| 温室 IoT 控制 | Codex 开发 IoT 控制 App | 外包开发 |
| 农业知识库客服 | RAG 风格聊天机器人 | 翻手册 / 问人 |

## 7 角度 × 3 钩子（写作复用）

### 角度 1：案例背书（OpenAI 官方）
- 钩子 1：不是"我朋友的故事"，是 OpenAI 官方案例
- 钩子 2：1500 亩 = 100 公顷 = 大型农场主
- 钩子 3：没有软件背景的农民做到了

### 角度 2：场景拆解（农业 AI 4 件套）
- 钩子 1：病虫害识别（多模态）
- 钩子 2：卫星遥感（主动监测）
- 钩子 3：IoT 温室控制 + 知识库客服

### 角度 3：路径迁移（ChatGPT → Codex）
- 钩子 1：ChatGPT 帮"学"，Codex 帮"做"
- 钩子 2：从问答到落地的范式升级
- 钩子 3：非程序员的 AI 工作流

### 角度 4：成本对比
- 钩子 1：不这么做 → 专门的外包团队 / 找开发公司 / 投入很大成本
- 钩子 2：这么做 → 思路打开 + 耐心 + 工具
- 钩子 3：普通人 ≠ 不能做复杂事

### 角度 5：研发思维 vs 普通用户思维
- 钩子 1：我们把工具当"工具"在研究
- 钩子 2：工具没有行业 / 人群限制
- 钩子 3：谁愿意尝试、谁能想到，谁就能用

### 角度 6：问题识别方法论
- 钩子 1：你工作里有没有每天重复出现、却没人解决的问题？
- 钩子 2：下一个被 Codex 改造的行业，不是高科技行业
- 钩子 3：找到那个重复 → 让 AI 接管

### 角度 7：反差 / 借势
- 钩子 1：研究生 AI 会什么 → 农民用 AI 改造行业
- 钩子 2："我们天天玩小龙虾，人家操控农场"
- 钩子 3：所以真正稀缺的不是 AI 能力，是真实业务问题
