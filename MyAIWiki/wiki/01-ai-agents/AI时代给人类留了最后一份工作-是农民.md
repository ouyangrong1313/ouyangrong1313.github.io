# AI时代给人类留了最后一份工作，是农民

## 核心结论

Brett Goldstein（microHQ 创始人，前 Google 认知科学家）提出：**人类最终的工作是 Context Farmer（语境农夫）**——不是做事，不是决策，而是给 AI 提供上下文，让业务之果自然生长。

**一句话：农民不产出水果，植物才产出。语境农夫做的，是让条件变好。**

## AMC：智能微型公司

Brett 提出 **Agentic Micro Company (AMC)** 概念，四要素：

| 要素 | 说明 |
|------|------|
| 极小全才团队 | 每个人都是「球员兼教练」，无专职分工壁垒 |
| 大量 Agent | 真正承担主要工作量，不只是辅助 |
| 定制软件 | 不买现成 SaaS，自己造自动化系统 |
| 公司大脑 | Agent 的上下文来源，软件的后端 |

**案例**：Jack Dorsey 把 Block 裁员 40%，重构成极小团队 + Agent 模式。硅谷创始人们几个月从零做到几千万收入。

## 公司大脑：上下文是王

Brett 观点：**哪怕是小模型，只要有更好的上下文，就能超越更大的模型。**

现有 Agent 的问题：量子物理学博士，但患有严重失忆症。

公司大脑在 AMC 里做三件事：
1. 作为自建软件的后端
2. 快速查找信息
3. 作为几百个 Agent 的蜂巢大脑

最好的记忆系统近似人类大脑：长期记忆、情节记忆、短期记忆。

## Context Farmer：人类最后的工作

随着模型越来越强，人类角色演进：

```
独立贡献者 → 微观管理者 → 总监 → 战略家 → 语境农夫
```

**为什么是"农夫"？**
- 农民不产出水果，植物才产出
- 农民做的是让条件变好：轮种、灌溉、日照
- 语境农夫监控上下文质量，从物理世界补充上下文，喂饱公司大脑

** Brett 的观察**：那些做得好的人，都有充裕的空闲时间——不忙，只是在"耕种"。

## 万亿美元机会：可组合的上下文层

**类比**：1908 年特拉维夫是荒漠，以色列人发明滴灌，100 年后从卫星能看到绿洲。改变一切的，不是更好的植物，是更好的基础设施。

**Brett 论点**：智能革命不会被智能本身催化，而是被可组合的上下文催化。

历次技术革命的催化剂：
| 革命 | 催化剂 |
|------|--------|
| 印刷术 | 活字（可组合文字） |
| 工业革命 | 可互换零件 |
| 互联网 | HTTP 协议 |
| 智能革命 | 可组合上下文 |

**机会**：改造上下文层，让个人和全球的上下文都对所有人可及。

## 三类人的行动建议

### 建公司的人
AMC 框架值得认真对待。「几个人 + 大量 Agent + 公司大脑」和传统「招更多人、建更多层级」逻辑完全相反。Jack Dorsey 已在 Block 验证。

### 做 AI 产品的人
「公司大脑」方向有真实市场需求。现在 Agent 是「有智识无记忆」状态。还没有人造出「改变一切的灌溉基础设施」，这个空白真实存在。

### 个人用户
把「Context Farmer」内化进工作方式：
- 给 AI 你的背景、约束、过去的判断和风格
- 同一模型，上下文不同，输出质量天差地别
- **不需要等「全球大脑」建成，现在就可以开始做自己的语境农夫**
