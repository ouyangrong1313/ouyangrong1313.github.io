- **Message/** - 消息中心
- **Mine/** - 个人中心

## Native SDK 集成

### JNI 规范

与 Native SDK（C/C++）交互使用 JNI：

```java
// JNI 调用示例
public native int st_init();
public native int st_connect(String ip, int port);
public native int st_disconnect();

// 在 System.loadLibrary() 时加载
static {
    System.loadLibrary("tps");
}
```

### 线程模型

- JNI 回调在子线程
- UI 更新需切换到主线程：`runOnUiThread()` 或 `Handler`

## 编译构建

Gradle 配合 CMake 编译 Native 代码：

```groovy
android {
    externalNativeBuild {
        cmake {
            path "src/main/cpp/CMakeLists.txt"
        }
    }
}
```
