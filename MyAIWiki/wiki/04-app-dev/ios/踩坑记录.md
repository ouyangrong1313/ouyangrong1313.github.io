# Seetong-iOS 开发踩坑记录

## 内存管理

### MLeaksFinder 检测

项目使用 MLeaksFinder 检测内存泄漏，继承自：
- `UIViewController+MemoryLeak`
- `UIPageViewController+MemoryLeak`
- `UISplitViewController+MemoryLeak`

**注意**：在 `-dealloc` 中打印日志可能触发误报。

### Category 方法释放

Category 添加的属性不会自动释放，需手动处理：

```objc
// 错误：category中的属性不会自动释放
@implementation UIViewController (MyCategory)
@dynamic someProperty;
@end

// 正确：使用关联对象并主动释放
- (void)dealloc {
    objc_setAssociatedObject(self, @selector(someProperty), nil, OBJC_ASSOCIATION_RETAIN_NONATOMIC);
}
```

## Auto Layout

### Masonry 约束冲突

同一视图多个方向的约束可能冲突，调试方法：

```objc
// 打印约束
self.label.constraints.description
```

### iPad 适配

项目使用 `kIsFit` 系列宏处理特殊屏幕：

```objc
#define kIsFit (IS_IPAD || IS_40INCH_SCREEN || IS_35INCH_SCREEN || IS_320WIDTH_SCREEN)
```

## 网络层

### YTKRequest 缓存

`STNetworkManager` 继承自 YTKRequest，注意缓存策略：

```objc
// 清除缓存
[self.model clearCache];
```

### 请求超时

`STNetworkManager` 配置了默认超时时间，特殊请求需单独设置。

## 音视频

### 设备专用库

视频编解码库（如 faac、faad）是**设备专用库**，模拟器无法运行。

### 线程切换

音视频回调通常在子线程，需切换到主线程更新 UI：

```objc
dispatch_async(dispatch_get_main_queue(), ^{
    // 更新UI
});
```

## 国际化

### 语言切换

使用自定义语言切换，非系统语言：

```objc
#define STRING(key,common) [[NSBundle bundleWithPath:[[NSBundle mainBundle] pathForResource:[[NSUserDefaults standardUserDefaults] objectForKey:@"seetongLanguage"] ofType:@"lproj"]] localizedStringForKey:(key) value:nil table:@"Localizable"]
```

```objc
#if DEBUG
    #define NSLog(s, ... ) NSLog( @"[%@ in line %d] %@", [[NSString stringWithUTF8String:__FILE__] lastPathComponent], __LINE__, [NSString stringWithFormat:(s), ##__VA_ARGS__] )

```
