# Seetong-iOS 模块划分

## Classes 目录结构

```
Seetong_tps/Classes/
├── AppLaunch/         # App 启动初始化
├── Base/              # 基础类（基类、通用View）
├── Contractor/        # 承包商管理
├── DeepLink/          # 深度链接
├── Device/            # 设备管理
├── DeviceList/        # 设备列表
├── Login/             # 登录模块
├── Message/           # 消息中心
├── Mine/              # 我的（个人中心）
├── PlayBack/          # 录像回放
└── TPS/               # 旧代码（迁移中）
```

## 模块说明

### Login 登录模块

```
Login/
├── Controller/
│   ├── STLoginViewController        # 登录
│   ├── STCountryViewController      # 国家选择
│   └── STOneKeyRegisterViewController  # 一键注册
├── Model/
└── Services/
    └── STLoginService               # 登录服务
```

### Device 设备模块

```
Device/
├── AddDevice/           # 添加设备
├── DeviceConfig/        # 设备配置
│   ├── DeviceSetting/    # 设备设置
│   └── ...
├── DeviceList/          # 设备列表
├── DevicePlayer/        # 设备播放
├── VideoPreview/        # 视频预览
├── Firmware/            # 固件升级
└── GroupManager/        # 分组管理
```

### Message 消息模块

消息推送、告警消息处理。

### Mine 我的模块

用户个人中心、设置页。

### Contractor 承包商模块

针对承包商角色的特殊功能：
- 日志查询（Logquery）
- Ping Box
- 存储管理（Storage）

## Base 基础类

```
Base/
├── Conrtoller/          # 注：目录名有拼写错误（Conrtoller）
│   ├── BaseViewController       # 基础VC
│   ├── BaseListViewController   # 列表VC基类
│   ├── BaseNavigationController # 导航栏VC
│   └── STFullScreenVC           # 全屏VC
├── Model/
│   └── BaseModel               # 基础Model
└── View/
    ├── STEmptyView             # 空状态视图
    ├── STNoDataView            # 无数据视图
    ├── STTextField             # 文本输入框
    ├── STSheetTitleView         # Sheet标题
    ├── STRemindView             # 提醒视图
    ├── STEmptyDataView          # 空数据视图
    ├── STLinkTextView           # 链接文本
    ├── STAlarmView              # 告警视图
    └── STSelectionView/         # 选择器组件
```
