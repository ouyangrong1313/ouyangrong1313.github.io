# 聪明人用AI做的，是给自己出题

## 核心结论

**理解一件事的标志，不是复述定义，而是能让它变回一个故事**

- 定义是压缩：丢弃了维度、情绪和后果
- 故事是解压缩：还原了动机、时间、对抗与遗憾

## Amanda Askell 三步法

Anthropic 研究员 Amanda Askell 分享的快速入门陌生领域提示词：

1. 找一个"本科生不了解、研究生才懂"的小众概念
2. 用三段式故事阐释，**故事结束前绝不提概念名**
3. 最后揭晓名称，解释故事如何对应概念每个要素

**三个精妙设计**：
- 刻意设置难度：太通俗没启发，太艰深过于遥远
- 延迟命名：逼迫大脑进入"深度理解"
- 反向费曼：让你听不懂名字，却懂了逻辑本身

## 核心 Prompt

```
我想搞懂[X]，但你别直接告诉我定义。我们这样来：
1、先找一个跟[X]很相近、但更冷门的原理；
2、用三段话编一个故事，把这个原理演出来——故事里不许出现概念的名字，要有冲突，要有压力，让我感受到当事人在做什么样的取舍；
3、故事讲完先别揭晓。先问我："你觉得这个故事在讲什么道理？" 等我答完，再告诉我[X]叫什么；
4、最后回头对照——故事里哪个情节对应了概念的哪一个核心要素，一条条说给我听。
```

## 关键洞察

> 平庸的问题只能得到平庸的答案。AI 的训练分布决定了它的"中心位置"永远是平庸的。对聪明人而言，你的问题必须足够具体、足够偏离轴心。

> 你不是"懂了"贝叶斯更新，你是在那个不断修正判断的侦探身上，经历了一次贝叶斯更新。**此为触感，而非定义**。

**聪明人的秘密**：用AI训练自己的大模型——你的大脑。
