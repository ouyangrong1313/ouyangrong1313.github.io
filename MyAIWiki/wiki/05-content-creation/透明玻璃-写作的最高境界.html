知识节点(8 个)

  • 透明-玻璃-核心定义:好的文字像透明玻璃——读者透过文字看到窗外的景致,以至于意识不到玻璃的存在;反例是有色玻璃——外表光鲜但读者需穿越层层”玻璃”才能勉强看到一点内容,阅读体验如雾里看花
  • 精准-特征:每个字都恰到好处,没有冗余,没有废话,像”池塘生春草”五个字字字珠玑、无可增减;王国维”不隔”标准——没有典故、没有藻饰、没有感叹号、没有可有可无的字
  • 凝练-特征:用最少文字传达最多信息;类比金刚石——体积小、密度大、硬度高;毛泽东《为人民服务》不足千字没有一个废字,邓小平南方讲话 7500 字还是 4 次讲话综合
  • 自然-特征:看似信手拈来,实则精心打磨;“清水出芙蓉,天然去雕饰”;读者读起来行云流水,完全感受不到”写作”的痕迹——阿西莫夫坦言”易读难写”
  • 三阶段-简单-绚丽-简单:刘军强指出写作脉络 = 简单 → 绚丽 → 简单;未经历绚丽的简单 = 简陋(功力不足);经历绚丽后的简单 = 透明(功力深厚);类比书法:稚嫩朴实 → 花团锦簇 → 平淡无奇实则功力深厚;看山还是山的第三重境界
  • 减法-思维-删三样:删冗余(铺垫太长/背景太杂/例子太多/论证重复,问”删掉影响核心思想表达吗?”)、删华丽装饰(形容词/副词/修饰语,例:”他非常非常生气,愤怒到了极点” → “他摔门而去”)、删陈词滥调(“众所周知”“不言而喻”“具有重要意义”等空洞表达)
  • 工匠-精神-多稿修改:理顺逻辑(核心观点/几个层次/层次关系先想清楚)、推敲字句(能用一字不用二字,能用动词不用形容词,能用具体事物不用抽象概念,例:”他感到很失望” → “他垂下手,指尖的温度一点点凉下去”)、多次修改(阿西莫夫/海明威/鲁迅都是多稿高手;第 1 稿给自己理思路,第 2 稿给读者调结构,第 3 稿给”敌人”删漏洞,第 4 稿+ 字字到位)
  • 精准表达-5-技巧:动作替状态(状态词抽象,读者需”理解”;动作具体,读者可”看见”)、具象替抽象(“春天来了万物复苏” → “池塘生春草”)、对话替叙述(对话透明,叙述隔膜)、短句替长句(长短结合错落有致,形成阅读节奏)、删去过渡词(“首先/其次/总而言之”等标签往往是简洁短文中的累赘)

正文要点(4 段)

一、透明玻璃的核心定义

“好的文字就像窗户上的透明玻璃,读者透过玻璃可以清晰地看到窗外的景致,以至于意识不到玻璃的存在。”

王国维”不隔”= 文字与读者之间没有隔阂,思想与情感直接呈现。

二、3 大特征与反例

特征 一句话 案例
精准 字字珠玑 “池塘生春草” 5 个字
凝练 金刚石级密度 《为人民服务》不足千字无废字
自然 看似信手拈来实则精心打磨 “清水出芙蓉,天然去雕饰”

反例(有色玻璃):过度修饰、堆砌辞藻、滥用典故、概念轰炸——读者穿越层层玻璃,如雾里看花。

三、为什么追求透明精炼

  • 信息过载时代必然选择——读者需要”直抵人心的真知灼见”,不是”华丽的废话”
  • 文章使命的回归——文字使命是”连接人心、传递思想”;文字本身”抢戏”=本末倒置
  • 深度思想需简洁表达——爱因斯坦:”不能简单解释它,说明你还没真正理解它”
  • 避免”简陋”陷阱——透明 ≠ 简陋;未经历绚丽的”简单” = 简陋(功力不足);经历绚丽后的”简单” = 透明(功力深厚)

四、4 步骤落地

  1. 减法思维——删冗余 + 删华丽 + 删陈词
  2. 工匠精神——理逻辑 + 推敲字句 + 多稿修改(4 稿以上)
  3. 遵循规律——大量阅读 → 刻意练习(绚丽)→ 返璞归真(透明)
  4. 精准表达——5 技巧:动作/具象/对话/短句/删过渡词

5 个对 Seetong 团队可借鉴动作

  1. PD 文档用”透明玻璃”标准评审——每段/每句/每字问”删掉影响核心思想吗?”,避免 AI 加冗余
  2. TAPD 需求附”3 句摘要”——精准 + 凝练:核心任务 / 用户场景 / 验收标准,每项 1 句
  3. code review 检查”华丽装饰”——AI 写的注释里的”显而易见”“显然”“不言而喻”全部删掉
  4. 写 SKILL.md 用”动作/具象”代”状态/抽象”——避免”应该”“需要”等抽象状态词
  5. 设”4 稿修改”硬约束——模仿文章建议(自己稿 / 读者稿 / 敌人稿 / 终稿)
