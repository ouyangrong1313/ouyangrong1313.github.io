# 文章结构库

## 用途

积累不同类型文章的结构模板，方便套用。

## 结构类型

### AIDA 结构
- Attention（注意力）
- Interest（兴趣）
- Desire（渴望）
- Action（行动）
- 适用：招生文案、销售页

### 空雨伞结构
- 天空（事实）
- 要不要带伞（意义）
- 带伞（行动）
- 适用：观点表达、建议类文章

### 方案结构
- 痛点
- 解药
- 具体做法
- 行动号召
- 适用：教程、指南

### 叙事结构
- 背景
- 冲突
- 高潮
- 结局
- 适用：案例分享、故事类
