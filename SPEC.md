# 童年跳房子大挑战 - 游戏规格说明书

## 1. 项目概述

- **项目名称**: 童年跳房子大挑战
- **项目类型**: H5 休闲怀旧小游戏
- **核心功能**: 复刻90年代童年跳房子游戏，点击控制跳跃，踩格子得分
- **目标用户**: 90后用户，微信手机端用户

## 2. 技术规范

### 技术栈
- 纯 HTML5 + CSS3 + JavaScript
- Canvas 2D 渲染
- LocalStorage 本地存储
- 无任何第三方依赖

### 屏幕适配
- 响应式设计，适配各种手机屏幕
- 基于 375px 宽度设计，缩放至全屏
- 设计稿尺寸: 375 x 667 (iPhone 6/7/8)

## 3. 视觉规格

### 配色方案
- 背景色: #F5E6D3 (米黄复古)
- 主色调: #FF6B6B (珊瑚红)
- 次色调: #4ECDC4 (薄荷绿)
- 强调色: #FFE66D (明黄色)
- 文字色: #2C3E50 (深灰蓝)
- 格子颜色: #E8D5B7 (浅米色)
- 粉笔线: #FFFFFF (白色)

### 字体
- 主字体: system-ui, -apple-system, "PingFang SC", "Microsoft YaHei", sans-serif
- 装饰字体: 粗体圆角

### 视觉效果
- 粉笔线条质感
- 复古地面纹理
- 手绘风格元素
- 简易弹跳动画
- 粒子特效(跳跃/得分)

## 4. 游戏玩法规格

### 游戏规则
- 单局时长: 60秒
- 格子布局: 经典跳房子 10 格
- 跳跃方式: 全屏点击/触摸
- 得分规则: 踩中格子 +10分
- 扣分规则: 踩空 -5分
- 跳跃节奏: 自动向右移动，玩家控制跳跃时机

### 角色
- 玩家: 简约卡通小人
- 移动: 自动从左向右移动
- 跳跃: 点击屏幕，小人跳跃

## 5. 页面规格

### 页面列表
1. **首页** - 游戏入口
2. **规则页** - 游戏说明
3. **游戏页** - 主游戏界面
4. **结算页** - 成绩展示+分享
5. **好友PK页** - 排行榜对比

### 首页元素
- 游戏标题(手绘风格)
- "开始闯关"按钮
- "好友PK"按钮
- "游戏规则"按钮
- 最高分显示

### 规则页元素
- 返回按钮
- 规则说明文字
- 示意图

### 游戏页元素
- 计时器(右上角)
- 当前分数(左上角)
- 跳房子地面
- 玩家小人
- 格子方块

### 结算页元素
- "再来一局"按钮
- "分享给好友"按钮
- "好友PK"按钮
- 本次得分
- 最高分记录
- 分享文案

### 好友PK页元素
- 我的分数
- 对比分数(预设/本地)
- 返回按钮

## 6. 音效规格

- 跳跃音效: 短促"滴"
- 得分音效: 清脆"叮"
- 失分音效: 低沉"咚"
- 背景: 无(安静)

## 7. 数据存储

### LocalStorage 结构
```javascript
{
  "hopscotch_highscore": number,    // 最高分
  "hopscotch_mypk_score": number,  // 我的PK分数
  "hopscotch_friend_score": number  // 好友分数(预设)
}
```
